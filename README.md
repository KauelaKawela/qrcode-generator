# 📱 Simple QR Generator

> Modern, hızlı ve kullanımı kolay bir QR kod oluşturucu.

🔗 **Canlı Demo:** [kauelakawela.github.io/qrcode-generator](https://kauelakawela.github.io/qrcode-generator)

---

## ✨ Özellikler

- 🔤 Herhangi bir metin veya URL için anında QR kod oluşturma
- 💾 Oluşturulan QR kodu PNG formatında indirme
- ⚡ Hızlı ve hafif — sunucu gerektirmez
- 🎨 Modern ve sade arayüz

---

## 🚀 Kullanım

1. [Demo sayfasını](https://kauelakawela.github.io/qrcode-generator) ziyaret edin
2. Metin veya URL girin
3. **"QR Kod Oluştur"** butonuna tıklayın
4. İsterseniz **"İndir (PNG)"** ile QR kodu kaydedin

---

## 🛠️ Kurulum

Projeyi yerel ortamınızda çalıştırmak için:

```bash
# Repoyu klonlayın
git clone https://github.com/KauelaKawela/qrcode-generator.git

# Proje klasörüne girin
cd qrcode-generator

# index.html dosyasını bir tarayıcıda açın
open index.html
```

> Herhangi bir bağımlılık kurulumu gerekmez — sadece bir tarayıcı yeterlidir.
